Detailed Description

Provides interface to Solve system of dipoles.

Allows to construct and Solve system of equations for the selected mathematical model.

Can accept various input formats for coordinates and solutions to crete system,

This class implements more verbose solution algorithm. It uses quasi symmetricity of block matrix

to cut computation i half.

to Solve it, to Generate direction graph for magnetic field.
